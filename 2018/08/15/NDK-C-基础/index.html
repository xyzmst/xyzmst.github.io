Q:C基础.001
动态内存申请
A:
malloc 没有初始化内存的内容,一般调用函数memset来初始化这部分的内存空间.
void jj = malloc(110241024);
size_t k = 11024*1024;
memset(jj,0,k);//指向内存 初始化 0 长度 k
calloc 申请内存并将初始化内存数据为NULL.
int *pn = (int)calloc(10, sizeof(int));
相当于
int pn = (int)malloc(10, sizeof(int));
memset(pn,0,sizeof(int)10);
realloc
对malloc申请的内存进行大小的调整.
特别的： alloca 在栈申请内存,因此无需释放.int *p = (int *)alloca(sizeof(int) * 10);
char *a = (char)malloc(10);
realloc(a,20);

Q:C基础.005
解引用
A:
解析并返回内存地址中保存的值
int i = 10;
int p = &i;
//解引用
//p指向一个内存地址，使用解出这个地址的值 即为 10
int pv = *p;
//修改地址的值,则i值也变成100
//为解引用的结果赋值也就是为指针所指的内存赋值
*p = 100;

Q:C基础.007
数组和指针
A:
在c语言中，指针和数组名都表示地址
1、数组是一块内存连续的数据。
2、指针是一个指向内存空间的变量

//数组指针
//二维数组类型是 int (p)[x]
int array[2][3] = { {11,22,33},{44,55,66} };
//也可以 int array[2][3] = { 11,22,33 ,44,55,66 };
//array1 就是一个 int[3] 类型的指针
int (*array1)[3] = array;
//怎么取 55 ？
//通过下标
array[1][1] == array1[1][1]
//通过解引用
int i = *((array1 + 1) + 1);
//拆分
//1、 指针偏移 因为array1的类型是3个int的数组 所以 +1 移动了12位
array1 + 1
//2、获得{44,55,66}数组 ((array1 + 1))[1] = 55
((array1 + 1)
//3、对数组执行 +/- 相当于隐式的转为指针
//获得 55 的地址 再解地址
*(array1 + 1) + 1

A:

int a = 10;

printf(“%d\n”, *j);

Q:C基础.010
可变参数
A:
与Java一样，C当中也有可变参数
va_start 第二个参数，传什么都无所谓，必须传
例子
#include <stdarg.h>
int add(int num, …)
{
va_list valist;
int sum = 0;
// 初始化 valist指向第一个可变参数 (…)
va_start(valist, num);
for (size_t i = 0; i < num; i++)
{
//访问所有赋给 valist 的参数
int j = va_arg(valist, int);
printf(“%d\n”, j);
sum += j;
}
//清理为 valist 内存
va_end(valist);
return sum;
}
