Freeline使用

最后更新:2016-12-25 阅读时间: 1 min 阅读量:

吐个槽吧

android 运行速度是越来越坑，小闪电(instant run)的各种bug，小火箭(jrebel android)的就那么一点试用时间，尝试了下freeline，还是蛮不错的

配置

最外层build.gradle 加入

dependencies {
        classpath 'com.antfortune.freeline:gradle:0.8.4'

    }

moudle下的build.gradle

apply plugin: 'com.antfortune.freeline'

freeline {
        hack true
        productFlavor "develop"
//        autoDependency false
    }

    productFlavors {
        normal {

        }

        develop {
        }
    }

初始化命令

./gradlew initFreeline -Pmirror

坑点

module中的 productFlavors 只能有一个渠道

freeline {
       hack true
       productFlavor 'your-flavor'
//        autoDependency false
   }

your-flavor 一定要是productFlavors 真实的渠道
否则报：
read project fail

Execution failed for task ‘:checkBeforeCleanBuild’.

freeline init慢的问题

Windows[CMD]: gradlew initFreeline -Pmirror
Linux/Mac: ./gradlew initFreeline -Pmirror
